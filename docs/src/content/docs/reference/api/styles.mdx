<div class="api-table">
| API | Description |
| --- | ----------- |
| [`header`](header) | General Heading style, e.g. `help_heading`. |
| [`error`](error) | Error heading. |
| [`usage`](usage) | Usage heading. |
| [`literal`](literal) | Literal command-line syntax, e.g. `--help` |
| [`placeholder`](placeholder) | Descriptions within command-line syntax, e.g. `value_name` |
| [`valid`](valid) | Highlight suggested usage. |
| [`invalid`](invalid) | Highlight invalid usage. |
| [`context`](context) | Highlight all specified contexts, e.g. `[default: false]`. |
| [`context_value`](context_value) | Highlight values within all of the context, e.g. the `false` in `[default: false]`. |
