# `value_parser` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.value_parser" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Specify the typed behavior of the argument.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "raw"
[args]
color = { long = "color", value-parser = ["always", "auto", "never"], default-value = "auto" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: raw
args:
  color:
    long: color
    value-parser:
    - always
    - auto
    - never
    default-value: auto
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "raw",
  "args": {
    "color": {
      "long": "color",
      "value-parser": [
        "always",
        "auto",
        "never"
      ],
      "default-value": "auto"
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog --color auto
```

### Output

```bash
claptrap_color='auto'
```
