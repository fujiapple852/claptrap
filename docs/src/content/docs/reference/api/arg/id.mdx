# `id` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.id" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Set the identifier used for referencing this argument in the clap API.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"
[args]
my-arg-id = { long = "foo", action = "set" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  my-arg-id:
    long: foo
    action: set
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "my-arg-id": {
      "long": "foo",
      "action": "set"
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog --foo file.toml
```

### Output

```bash
claptrap_my-arg-id='file.toml'
```
