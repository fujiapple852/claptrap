# `requires_if` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.requires_if" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Require another argument if this arg matches the ArgPredicate.

# Examples
## Example 1

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"
[args]
config = { long = "config", action = "set", requires-if = { arg = "other", value = "my.cfg" } }
other = {}
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  config:
    long: config
    action: set
    requires-if:
      arg: other
      value: my.cfg
  other: {}
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "config": {
      "long": "config",
      "action": "set",
      "requires-if": {
        "arg": "other",
        "value": "my.cfg"
      }
    },
    "other": {}
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog --config some.cfg
```

### Output

```bash
claptrap_config='some.cfg'
```
