# `default_values` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.default_values" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Value for the argument when not present.

# Examples
## Example 1

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"
[args]
opt = { long = "myopt", num-args = 2, default-values = ["myval1", "myval2"] }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  opt:
    long: myopt
    num-args: 2
    default-values:
    - myval1
    - myval2
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "opt": {
      "long": "myopt",
      "num-args": 2,
      "default-values": [
        "myval1",
        "myval2"
      ]
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog
```

### Output

```bash
claptrap_opt=('myval1' 'myval2')
```
