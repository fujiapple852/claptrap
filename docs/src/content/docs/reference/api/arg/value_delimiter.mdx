# `value_delimiter` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.value_delimiter" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Allow grouping of multiple values via a delimiter.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"
[args]
config = { short = 'c', long = "config", value-delimiter = "," }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  config:
    short: c
    long: config
    value-delimiter: ','
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "config": {
      "short": "c",
      "long": "config",
      "value-delimiter": ","
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog --config=val1,val2,val3
```

### Output

```bash
claptrap_config=('val1' 'val2' 'val3')
```
