# `help` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.help" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Sets the description of the argument for short help (`-h`).

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"
[args]
cfg = { long = "config", help = "Some help text describing the --config arg" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  cfg:
    long: config
    help: Some help text describing the --config arg
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "cfg": {
      "long": "config",
      "help": "Some help text describing the --config arg"
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog --help
```

### Output

```bash
Usage: prog [OPTIONS]

Options:
      --config <cfg>  Some help text describing the --config arg
  -h, --help          Print help
```
