# `requires` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.requires" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Sets an argument that is required when this one is present.

# Examples
## Example 1

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"
[args]
cfg = { long = "config", requires = "input", action = "set" }
input = {}
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  cfg:
    long: config
    requires: input
    action: set
  input: {}
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "cfg": {
      "long": "config",
      "requires": "input",
      "action": "set"
    },
    "input": {}
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog
```

### Output

```bash

```
