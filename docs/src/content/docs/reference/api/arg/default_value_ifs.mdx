# `default_value_ifs` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.default_value_ifs" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Specifies multiple values and conditions in the same manner as `default_value_if`.

# Examples
## Example 1

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"
[args]
flag = { long = "flag", action = "set-true" }
opt = { long = "opt", action = "set" }
other = { long = "other", default-value-ifs = [{ arg = "flag", value = "true", default = "default" }, { arg = "opt", value = "channel", default = "chan" }] }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  flag:
    long: flag
    action: set-true
  opt:
    long: opt
    action: set
  other:
    long: other
    default-value-ifs:
    - arg: flag
      value: 'true'
      default: default
    - arg: opt
      value: channel
      default: chan
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "flag": {
      "long": "flag",
      "action": "set-true"
    },
    "opt": {
      "long": "opt",
      "action": "set"
    },
    "other": {
      "long": "other",
      "default-value-ifs": [
        {
          "arg": "flag",
          "value": "true",
          "default": "default"
        },
        {
          "arg": "opt",
          "value": "channel",
          "default": "chan"
        }
      ]
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog --opt channel
```

### Output

```bash
claptrap_opt='channel'
claptrap_flag='false'
claptrap_other='chan'
```
