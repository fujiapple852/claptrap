# `hide_short_help` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.hide_short_help" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Hides an argument from short help (`-h`).

# Examples
## Example 1

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"
[args]
cfg = { long = "config", hide-short-help = true, help = "Some help text describing the --config arg" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  cfg:
    long: config
    hide-short-help: true
    help: Some help text describing the --config arg
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "cfg": {
      "long": "config",
      "hide-short-help": true,
      "help": "Some help text describing the --config arg"
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog -h
```

### Output

```bash
Usage: prog [OPTIONS]

Options:
  -h, --help  Print help (see more with '--help')
```
