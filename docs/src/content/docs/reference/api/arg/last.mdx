# `last` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.last" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

This arg is the last, or final, positional argument (i.e. has the highest index) and is only able to be accessed via the `--` syntax (i.e. `$ prog args -- last_arg`).

# Examples
## Example 1

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"
[args]
first = {}
second = {}
third = { action = "set", last = true }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  first: {}
  second: {}
  third:
    action: set
    last: true
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "first": {},
    "second": {},
    "third": {
      "action": "set",
      "last": true
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog one -- three
```

### Output

```bash
claptrap_first='one'
claptrap_third='three'
```
