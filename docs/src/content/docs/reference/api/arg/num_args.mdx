# `num_args` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.num_args" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Specifies the number of arguments parsed per occurrence.

# Examples
## Example 1

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"
[args]
mode = { long = "mode", default-missing-value = "slow", default-value = "plaid", num-args = "optional" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  mode:
    long: mode
    default-missing-value: slow
    default-value: plaid
    num-args: optional
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "mode": {
      "long": "mode",
      "default-missing-value": "slow",
      "default-value": "plaid",
      "num-args": "optional"
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog --mode fast
```

### Output

```bash
claptrap_mode='fast'
```
