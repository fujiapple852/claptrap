# `allow_hyphen_values` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.allow_hyphen_values" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Allows values which start with a leading hyphen (`-`)

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"
[args]
pat = { long = "pattern", action = "set", allow-hyphen-values = true }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  pat:
    long: pattern
    action: set
    allow-hyphen-values: true
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "pat": {
      "long": "pattern",
      "action": "set",
      "allow-hyphen-values": true
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog --pattern -file
```

### Output

```bash
claptrap_pat='-file'
```
