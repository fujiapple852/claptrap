# `visible_short_alias` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.visible_short_alias" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Add an alias, which functions as a visible short flag.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"
[args]
test = { long = "test", visible-short-alias = 't', action = "set" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  test:
    long: test
    visible-short-alias: t
    action: set
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "test": {
      "long": "test",
      "visible-short-alias": "t",
      "action": "set"
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog -t coffee
```

### Output

```bash
claptrap_test='coffee'
```
