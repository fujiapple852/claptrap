# `default_value_if` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.default_value_if" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Specifies the value of the argument if arg has been used at runtime.

# Examples
## Example 1

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"
[args]
opt = { long = "opt", action = "set" }
other = { long = "other", default-value-if = { arg = "opt", value = "special", default = "default" } }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  opt:
    long: opt
    action: set
  other:
    long: other
    default-value-if:
      arg: opt
      value: special
      default: default
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "opt": {
      "long": "opt",
      "action": "set"
    },
    "other": {
      "long": "other",
      "default-value-if": {
        "arg": "opt",
        "value": "special",
        "default": "default"
      }
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog --opt special
```

### Output

```bash
claptrap_opt='special'
claptrap_other='default'
```
