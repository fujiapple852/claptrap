# `hide` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.hide" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Hide this argument from the generated **help** text.

## Example

#### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"
[args]
cfg = { long = "config", hide = true }
```
    </TabItem>
    <TabItem label="YAML">
```toml
name: prog
args:
  cfg:
    long: config
    hide: true
```
    </TabItem>
    <TabItem label="JSON">
```toml
{
  "name": "prog",
  "args": {
    "cfg": {
      "long": "config",
      "hide": true
    }
  }
}
```
    </TabItem>
</Tabs>

#### Input

```bash
prog --help
```

#### Output

```bash
Usage: prog

Options:
  -h, --help  Print help
```
