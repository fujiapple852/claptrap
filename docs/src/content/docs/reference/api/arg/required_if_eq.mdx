# `required_if_eq` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.required_if_eq" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

This argument is required only if the specified arg is present at runtime and its value equals val.

# Examples
## Example 1

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"
[args]
cfg = { long = "config", action = "set", required-if-eq = { arg = "other", value = "special" } }
other = { long = "other", action = "set" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  cfg:
    long: config
    action: set
    required-if-eq:
      arg: other
      value: special
  other:
    long: other
    action: set
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "cfg": {
      "long": "config",
      "action": "set",
      "required-if-eq": {
        "arg": "other",
        "value": "special"
      }
    },
    "other": {
      "long": "other",
      "action": "set"
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog --other not-special
```

### Output

```bash
claptrap_other='not-special'
```
