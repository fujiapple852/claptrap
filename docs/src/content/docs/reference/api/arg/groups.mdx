# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"
[args]
debug = { long = "debug", action = "set-true", groups = ["mode", "verbosity"] }
verbose = { long = "verbose", action = "set-true", groups = ["mode", "verbosity"] }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  debug:
    long: debug
    action: set-true
    groups:
    - mode
    - verbosity
  verbose:
    long: verbose
    action: set-true
    groups:
    - mode
    - verbosity
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "debug": {
      "long": "debug",
      "action": "set-true",
      "groups": [
        "mode",
        "verbosity"
      ]
    },
    "verbose": {
      "long": "verbose",
      "action": "set-true",
      "groups": [
        "mode",
        "verbosity"
      ]
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog --verbose
```

### Output

```bash
claptrap_verbose='true'
```
