# `hide_env_values` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.hide_env_values" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Do not display in help any values inside the associated ENV variables for the argument.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "connect"
[args]
host = { long = "host", env = "CONNECT", action = "set", hide-env-values = true }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: connect
args:
  host:
    long: host
    env: CONNECT
    action: set
    hide-env-values: true
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "connect",
  "args": {
    "host": {
      "long": "host",
      "env": "CONNECT",
      "action": "set",
      "hide-env-values": true
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
connect --help
```

### Output

```bash
Usage: connect [OPTIONS]

Options:
      --host <host>  [env: CONNECT]
  -h, --help         Print help
```
