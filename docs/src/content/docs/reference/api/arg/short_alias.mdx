# `short_alias` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.short_alias" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Add an alias, which functions as a hidden short flag.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"
[args]
test = { short = 't', short-alias = 'e', action = "set" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  test:
    short: t
    short-alias: e
    action: set
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "test": {
      "short": "t",
      "short-alias": "e",
      "action": "set"
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog -e cool
```

### Output

```bash
claptrap_test='cool'
```
