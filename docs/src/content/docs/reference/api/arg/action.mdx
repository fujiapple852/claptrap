# `action` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.action" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Specify how to react to an argument when parsing it.

# Examples
## Example 1

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"
[args]
flag = { long = "flag", action = "append" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  flag:
    long: flag
    action: append
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "flag": {
      "long": "flag",
      "action": "append"
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog --flag value1 --flag value2
```

### Output

```bash
claptrap_flag=('value1' 'value2')
```
