# `hide_env` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.hide_env" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Do not display in help the environment variable name.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"
[args]
mode = { long = "mode", env = "MODE", action = "set", hide-env = true }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  mode:
    long: mode
    env: MODE
    action: set
    hide-env: true
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "mode": {
      "long": "mode",
      "env": "MODE",
      "action": "set",
      "hide-env": true
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog --help
```

### Output

```bash
Usage: prog [OPTIONS]

Options:
      --mode <mode>  
  -h, --help         Print help
```
