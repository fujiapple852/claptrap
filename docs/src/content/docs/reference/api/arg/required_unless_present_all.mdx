# `required_unless_present_all` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.required_unless_present_all" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Sets this arg as required unless all of the specified arguments are present at runtime.

# Examples
## Example 1

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"
[args]
cfg = { long = "config", required-unless-present-all = ["dbg", "infile"], action = "set" }
dbg = { long = "debug", action = "set-true" }
infile = { short = 'i', action = "set" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  cfg:
    long: config
    required-unless-present-all:
    - dbg
    - infile
    action: set
  dbg:
    long: debug
    action: set-true
  infile:
    short: i
    action: set
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "cfg": {
      "long": "config",
      "required-unless-present-all": [
        "dbg",
        "infile"
      ],
      "action": "set"
    },
    "dbg": {
      "long": "debug",
      "action": "set-true"
    },
    "infile": {
      "short": "i",
      "action": "set"
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog --debug -i file
```

### Output

```bash
claptrap_dbg='true'
claptrap_infile='file'
```
