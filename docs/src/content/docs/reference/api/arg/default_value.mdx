# `default_value` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.default_value" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Value for the argument when not present.

# Examples
## Example 1

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"
[args]
opt = { long = "myopt", default-value = "myval" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  opt:
    long: myopt
    default-value: myval
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "opt": {
      "long": "myopt",
      "default-value": "myval"
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog
```

### Output

```bash
claptrap_opt='myval'
```
