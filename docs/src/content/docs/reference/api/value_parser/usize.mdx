Parse `usize` values.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"

[args]
foo = { long = "foo", value-parser = ":usize:" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  foo:
    long: foo
    value-parser: ':usize:'
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "foo": {
      "long": "foo",
      "value-parser": ":usize:"
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog --foo 18446744073709551615
```

### Output

```bash
claptrap_foo='18446744073709551615'
```
