Parse `f32` values.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"

[args]
foo = { long = "foo", value-parser = ":f32:" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  foo:
    long: foo
    value-parser: ':f32:'
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "foo": {
      "long": "foo",
      "value-parser": ":f32:"
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog --foo 3.14
```

### Output

```bash
claptrap_foo='3.14'
```
