# `boolish` <LinkButton href="https://docs.rs/clap/latest/clap/value_parser/fn.boolish.html" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Parse bool-like string values.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"

[args]
foo = { long = "foo", value-parser = ":boolish:" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  foo:
    long: foo
    value-parser: ':boolish:'
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "foo": {
      "long": "foo",
      "value-parser": ":boolish:"
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog --foo yes
```

### Output

```bash
claptrap_foo='true'
```
