Parse string values.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"

[args]
foo = { long = "foo", value-parser = ":string:" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  foo:
    long: foo
    value-parser: ':string:'
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "foo": {
      "long": "foo",
      "value-parser": ":string:"
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog --foo bar
```

### Output

```bash
claptrap_foo='bar'
```
