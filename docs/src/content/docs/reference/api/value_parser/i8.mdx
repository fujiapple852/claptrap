Parse `i8` values.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"

[args]
foo = { long = "foo", value-parser = ":i8:", allow-negative-numbers = true }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  foo:
    long: foo
    value-parser: ':i8:'
    allow-negative-numbers: true
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "foo": {
      "long": "foo",
      "value-parser": ":i8:",
      "allow-negative-numbers": true
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog --foo -128
```

### Output

```bash
claptrap_foo='-128'
```
