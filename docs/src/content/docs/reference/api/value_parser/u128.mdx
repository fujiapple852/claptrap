# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"

[args]
foo = { long = "foo", value-parser = ":u128:" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  foo:
    long: foo
    value-parser: ':u128:'
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "foo": {
      "long": "foo",
      "value-parser": ":u128:"
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog --foo 340282366920938463463374607431768211455
```

### Output

```bash
claptrap_foo='340282366920938463463374607431768211455'
```
