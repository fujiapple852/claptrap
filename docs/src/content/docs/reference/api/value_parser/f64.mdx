# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"

[args]
foo = { long = "foo", value-parser = ":f64:" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  foo:
    long: foo
    value-parser: ':f64:'
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "foo": {
      "long": "foo",
      "value-parser": ":f64:"
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog --foo 123456.78901234567
```

### Output

```bash
claptrap_foo='123456.78901234567'
```
