Parse `u32` values.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"

[args]
foo = { long = "foo", value-parser = ":u32:" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
args:
  foo:
    long: foo
    value-parser: ':u32:'
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "args": {
    "foo": {
      "long": "foo",
      "value-parser": ":u32:"
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog --foo 4294967295
```

### Output

```bash
claptrap_foo='4294967295'
```
