---
title: "ArgGroup"
description: "The ArgGroup API"
---

<div class="api-table">
| API | Description |
| --- | ----------- |
| [`id`](id) | Sets the group name. |
| [`args`](args) | Adds multiple arguments to this group by name. |
| [`multiple`](multiple) | Allows more than one of the Args in this group to be used. |
| [`required`](required) | Require an argument from the group to be present when parsing. |
| [`requires`](requires) | Specify an argument or group that must be present when this group is. |
| [`requires_all`](requires_all) | Specify arguments or groups that must be present when this group is. |
| [`conflicts_with`](conflicts_with) | Specify an argument or group that must not be present when this group is. |
| [`conflicts_with_all`](conflicts_with_all) | Specify arguments or groups that must not be present when this group is. |
</div>
