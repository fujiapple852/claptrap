# `color` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Command.html#method.color" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Sets when to color output.

# Examples
## Example 1

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "myprog"
color = "never"
[args]
cfg = { long = "config" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: myprog
color: never
args:
  cfg:
    long: config
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "myprog",
  "color": "never",
  "args": {
    "cfg": {
      "long": "config"
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
myprog --help
```

### Output

```bash
Usage: myprog [OPTIONS]

Options:
      --config <cfg>  
  -h, --help          Print help
```
