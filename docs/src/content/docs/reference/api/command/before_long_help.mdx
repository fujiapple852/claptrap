# `before_long_help` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Command.html#method.before_long_help" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Free-form help text for before auto-generated long help (`--help`).

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "myprog"
before-long-help = "Some verbose and long info I'd like to appear before the help info"
[args]
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: myprog
before-long-help: Some verbose and long info I'd like to appear before the help info
args: {}
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "myprog",
  "before-long-help": "Some verbose and long info I'd like to appear before the help info",
  "args": {}
}
```
  </TabItem>
</Tabs>

### Input

```bash
myprog --help
```

### Output

```bash
Some verbose and long info I'd like to appear before the help info


Usage: myprog

Options:
  -h, --help
          Print help (see a summary with '-h')
```
