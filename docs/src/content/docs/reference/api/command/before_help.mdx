# `before_help` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Command.html#method.before_help" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Free-form help text for before auto-generated short help (`-h`).

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "myprog"
before-help = "Some info I'd like to appear before the help info"
[args]
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: myprog
before-help: Some info I'd like to appear before the help info
args: {}
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "myprog",
  "before-help": "Some info I'd like to appear before the help info",
  "args": {}
}
```
  </TabItem>
</Tabs>

### Input

```bash
myprog -h
```

### Output

```bash
Some info I'd like to appear before the help info


Usage: myprog

Options:
  -h, --help  Print help
```
