# `disable_help_flag` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Command.html#method.disable_help_flag" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Disables `-h` and `--help` flag.

# Examples
## Example 1

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "myprog"
disable-help-flag = true
[args]
help = { long = "help", action = "help", help = "Print version" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: myprog
disable-help-flag: true
args:
  help:
    long: help
    action: help
    help: Print version
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "myprog",
  "disable-help-flag": true,
  "args": {
    "help": {
      "long": "help",
      "action": "help",
      "help": "Print version"
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
myprog -h
```

### Output

```bash
error: unexpected argument '-h' found

Usage: myprog

For more information, try '--help'.
```
