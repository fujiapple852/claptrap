# `visible_short_flag_alias` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Command.html#method.visible_short_flag_alias" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Add an alias, which functions as “visible” short flag subcommand.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "myprog"
[[subcommands]]
name = "test"
short-flag = 't'
visible-short-flag-alias = 'd'
[subcommands.args]
input = { required = false, help = "the file to add" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: myprog
subcommands:
- name: test
  short-flag: t
  visible-short-flag-alias: d
  args:
    input:
      required: false
      help: the file to add
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "myprog",
  "subcommands": [
    {
      "name": "test",
      "short-flag": "t",
      "visible-short-flag-alias": "d",
      "args": {
        "input": {
          "required": false,
          "help": "the file to add"
        }
      }
    }
  ]
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog -d file.txt
```

### Output

```bash
claptrap__subcommand='test'
claptrap_test_input='file.txt'
```
