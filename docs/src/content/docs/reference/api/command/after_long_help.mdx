# `after_long_help` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Command.html#method.after_long_help" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Free-form help text for after auto-generated long help (`--help`).

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "myprog"
term-width = 80
after-long-help = "Does really amazing things to great people... but be careful with -R, like, for real, be careful with this!"
[args]
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: myprog
term-width: 80
after-long-help: Does really amazing things to great people... but be careful with -R, like, for real, be careful with this!
args: {}
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "myprog",
  "term-width": 80,
  "after-long-help": "Does really amazing things to great people... but be careful with -R, like, for real, be careful with this!",
  "args": {}
}
```
  </TabItem>
</Tabs>

### Input

```bash
myprog --help
```

### Output

```bash
Usage: myprog

Options:
  -h, --help
          Print help (see a summary with '-h')

Does really amazing things to great people... but be careful with -R, like, for
real, be careful with this!
```
