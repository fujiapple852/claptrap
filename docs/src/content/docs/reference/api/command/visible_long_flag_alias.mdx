# `visible_long_flag_alias` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Command.html#method.visible_long_flag_alias" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Add an alias, which functions as a “visible” long flag subcommand.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "myprog"
[[subcommands]]
name = "test"
long-flag = "test"
visible-long-flag-alias = "testing"
[subcommands.args]
input = { required = false, help = "the file to add" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: myprog
subcommands:
- name: test
  long-flag: test
  visible-long-flag-alias: testing
  args:
    input:
      required: false
      help: the file to add
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "myprog",
  "subcommands": [
    {
      "name": "test",
      "long-flag": "test",
      "visible-long-flag-alias": "testing",
      "args": {
        "input": {
          "required": false,
          "help": "the file to add"
        }
      }
    }
  ]
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog --testing file.txt
```

### Output

```bash
claptrap__subcommand='test'
claptrap_test_input='file.txt'
```
