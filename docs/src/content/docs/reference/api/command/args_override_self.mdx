# `args_override_self` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Command.html#method.args_override_self" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Replace prior occurrences of arguments rather than error.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "cmd"
args-override-self = true
[args]
foo = { long = "foo" }
bar = { long = "bar" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: cmd
args-override-self: true
args:
  foo:
    long: foo
  bar:
    long: bar
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "cmd",
  "args-override-self": true,
  "args": {
    "foo": {
      "long": "foo"
    },
    "bar": {
      "long": "bar"
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
cmd --foo value --foo value
```

### Output

```bash
claptrap_foo='value'
```
