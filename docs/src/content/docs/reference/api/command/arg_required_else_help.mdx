# `arg_required_else_help` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Command.html#method.arg_required_else_help" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Exit gracefully if no arguments are present (e.g. `$ myprog`).

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "myprog"
arg-required-else-help = true
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: myprog
arg-required-else-help: true
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "myprog",
  "arg-required-else-help": true
}
```
  </TabItem>
</Tabs>

### Input

```bash
myprog
```

### Output

```bash
Usage: myprog

Options:
  -h, --help  Print help
```
