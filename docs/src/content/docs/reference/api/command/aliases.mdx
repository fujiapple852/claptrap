# `aliases` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Command.html#method.aliases" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Sets multiple hidden aliases to this subcommand.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "myprog"
[[subcommands]]
name = "test"
aliases = ["do-stuff", "do-tests", "tests"]
[subcommands.args]
input = { required = false, help = "the file to add" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: myprog
subcommands:
- name: test
  aliases:
  - do-stuff
  - do-tests
  - tests
  args:
    input:
      required: false
      help: the file to add
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "myprog",
  "subcommands": [
    {
      "name": "test",
      "aliases": [
        "do-stuff",
        "do-tests",
        "tests"
      ],
      "args": {
        "input": {
          "required": false,
          "help": "the file to add"
        }
      }
    }
  ]
}
```
  </TabItem>
</Tabs>

### Input

```bash
myprog do-tests
```

### Output

```bash
claptrap__subcommand='test'
```
