# `ignore_errors` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Command.html#method.ignore_errors" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Try not to fail on parse errors, like missing option values.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "cmd"
ignore-errors = true
[args]
config = { short = 'c', long = "config", value-name = "FILE", help = "Sets a custom config file" }
stuff = { short = 's', long = "stuff", value-name = "FILE", help = "Sets a custom stuff file" }
f = { short = 'f', help = "Flag" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: cmd
ignore-errors: true
args:
  config:
    short: c
    long: config
    value-name: FILE
    help: Sets a custom config file
  stuff:
    short: s
    long: stuff
    value-name: FILE
    help: Sets a custom stuff file
  f:
    short: f
    help: Flag
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "cmd",
  "ignore-errors": true,
  "args": {
    "config": {
      "short": "c",
      "long": "config",

      "help": "Sets a custom config file"
    },

      "short": "s",

```
  </TabItem>
</Tabs>

### Input

```bash
prog -c file -f -x
```

### Output

```bash
claptrap_config='file'
```
