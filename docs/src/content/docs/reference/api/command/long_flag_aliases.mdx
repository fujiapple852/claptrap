# `long_flag_aliases` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Command.html#method.long_flag_aliases" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Add aliases, which function as “hidden” long flag subcommands.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "myprog"
[[subcommands]]
name = "test"
long-flag = "test"
long-flag-aliases = ["testing", "testall", "test_all"]
[subcommands.args]
input = { required = false, help = "the file to add" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: myprog
subcommands:
- name: test
  long-flag: test
  long-flag-aliases:
  - testing
  - testall
  - test_all
  args:
    input:
      required: false
      help: the file to add
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "myprog",
  "subcommands": [
    {
      "name": "test",
      "long-flag": "test",
      "long-flag-aliases": [
        "testing",
        "testall",
        "test_all"
      ],
      "args": {
        "input": {
          "required": false,
          "help": "the file to add"
        }
      }
    }
  ]
}
```
  </TabItem>
</Tabs>

### Input

```bash
myprog --testing
```

### Output

```bash
claptrap__subcommand='test'
```
