# `infer_subcommands` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Command.html#method.infer_subcommands" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Allow partial matches of subcommand names and their aliases.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "myprog"
infer-subcommands = true
[[subcommands]]
name = "test"
[subcommands.args]
arg1 = { long = "arg1", action = "set-true" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: myprog
infer-subcommands: true
subcommands:
- name: test
  args:
    arg1:
      long: arg1
      action: set-true
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "myprog",
  "infer-subcommands": true,
  "subcommands": [
    {
      "name": "test",
      "args": {
        "arg1": {
          "long": "arg1",
          "action": "set-true"
        }
      }
    }
  ]
}
```
  </TabItem>
</Tabs>

### Input

```bash
myprog te --arg1
```

### Output

```bash
claptrap__subcommand='test'
claptrap_test_arg1='true'
```
