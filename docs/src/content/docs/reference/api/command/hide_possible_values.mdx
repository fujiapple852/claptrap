# `hide_possible_values` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Command.html#method.hide_possible_values" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Tells clap not to print possible values when displaying help information.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "prog"
hide-possible-values = true
[args]
mode = { long = "mode", value-parser = ["fast", "slow"], action = "set" }
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: prog
hide-possible-values: true
args:
  mode:
    long: mode
    value-parser:
    - fast
    - slow
    action: set
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "prog",
  "hide-possible-values": true,
  "args": {
    "mode": {
      "long": "mode",
      "value-parser": [
        "fast",
        "slow"
      ],
      "action": "set"
    }
  }
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog --help
```

### Output

```bash
Usage: prog [OPTIONS]

Options:
      --mode <mode>  
  -h, --help         Print help
```
