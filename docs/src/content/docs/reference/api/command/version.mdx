# `version` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Command.html#method.version" icon="open-book" variant="minimal"></LinkButton>

<Badge text="since 0.1.0" size="small" />

Sets the version for the short version (`-V`) and help messages.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
```toml
name = "myprog"
version = "v0.1.24"
[args]
```
  </TabItem>
  <TabItem label="YAML">
```yaml
name: myprog
version: v0.1.24
args: {}
```
  </TabItem>
  <TabItem label="JSON">
```json
{
  "name": "myprog",
  "version": "v0.1.24",
  "args": {}
}
```
  </TabItem>
</Tabs>

### Input

```bash
prog -V
```

### Output

```bash
myprog v0.1.24
```
