# `alias` <LinkButton href="https://docs.rs/clap/latest/clap/builder/struct.PossibleValue.html#method.alias" icon="open-book" variant="minimal" />

<Badge text="since 0.1.0" size="small" />

Sets a hidden alias for this argument value.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
    ```toml
    name = "prog"
    [args]
    mode = { long = "mode", action = "set", value-parser = [{ name = "fast", alias = "speedy", help = "speedy" }, { name = "slow", help = "very slow", hide = true }, { name = "express", aliases = ["e", "exp"] }] }
    ```
  </TabItem>

  <TabItem label="YAML">
    ```yaml
    name: prog
    args:
      mode:
        long: mode
        action: set
        value-parser:
        - name: fast
          alias: speedy
          help: speedy
        - name: slow
          help: very slow
          hide: true
        - name: express
          aliases:
          - e
          - exp
    ```
  </TabItem>

  <TabItem label="JSON">
    ```json
    {
      "name": "prog",
      "args": {
        "mode": {
          "long": "mode",
          "action": "set",
          "value-parser": [
            {
              "name": "fast",
              "alias": "speedy",
              "help": "speedy"
            },
            {
              "name": "slow",
              "help": "very slow",
              "hide": true
            },
            {
              "name": "express",
              "aliases": [
                "e",
                "exp"
              ]
            }
          ]
        }
      }
    }
    ```
  </TabItem>
</Tabs>

### Input

```bash
prog --mode speedy
```

### Output

```bash
claptrap_mode='speedy'
```
