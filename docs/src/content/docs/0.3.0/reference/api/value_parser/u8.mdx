# `u8` <LinkButton href="https://docs.rs/clap/latest/clap/value_parser/fn.u8.html" icon="open-book" variant="minimal" />

<Badge text="since 0.1.0" size="small" />

Parse `u8` values.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
    ```toml
    name = "prog"

    [args]
    foo = { long = "foo", value-parser = ":u8:" }
    ```
  </TabItem>

  <TabItem label="YAML">
    ```yaml
    name: prog
    args:
      foo:
        long: foo
        value-parser: ':u8:'
    ```
  </TabItem>

  <TabItem label="JSON">
    ```json
    {
      "name": "prog",
      "args": {
        "foo": {
          "long": "foo",
          "value-parser": ":u8:"
        }
      }
    }
    ```
  </TabItem>
</Tabs>

### Input

```bash
prog --foo 255
```

### Output

```bash
claptrap_foo='255'
```
