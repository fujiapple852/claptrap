# `falsey` <LinkButton href="https://docs.rs/clap/latest/clap/value_parser/fn.falsey.html" icon="open-book" variant="minimal" />

<Badge text="since 0.1.0" size="small" />

Parse false-like string values, everything else is true.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
    ```toml
    name = "prog"

    [args]
    foo = { long = "foo", value-parser = ":falsey:" }
    ```
  </TabItem>

  <TabItem label="YAML">
    ```yaml
    name: prog
    args:
      foo:
        long: foo
        value-parser: ':falsey:'
    ```
  </TabItem>

  <TabItem label="JSON">
    ```json
    {
      "name": "prog",
      "args": {
        "foo": {
          "long": "foo",
          "value-parser": ":falsey:"
        }
      }
    }
    ```
  </TabItem>
</Tabs>

### Input

```bash
prog --foo no
```

### Output

```bash
claptrap_foo='false'
```
