# `args` <LinkButton href="https://docs.rs/clap/latest/clap/struct.ArgGroup.html#method.args" icon="open-book" variant="minimal" />

<Badge text="since 0.1.0" size="small" />

Adds multiple arguments to this group by name.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
    ```toml
    name = "myprog"
    [groups]
    req_flags = { args = ["flag", "color"] }
    [args]
    flag = { short = 'f', action = "set-true" }
    color = { short = 'c', action = "set-true" }
    ```
  </TabItem>

  <TabItem label="YAML">
    ```yaml
    name: myprog
    groups:
      req_flags:
        args:
        - flag
        - color
    args:
      flag:
        short: f
        action: set-true
      color:
        short: c
        action: set-true
    ```
  </TabItem>

  <TabItem label="JSON">
    ```json
    {
      "name": "myprog",
      "groups": {
        "req_flags": {
          "args": [
            "flag",
            "color"
          ]
        }
      },
      "args": {
        "flag": {
          "short": "f",
          "action": "set-true"
        },
        "color": {
          "short": "c",
          "action": "set-true"
        }
      }
    }
    ```
  </TabItem>
</Tabs>

### Input

```bash
myprog -f
```

### Output

```bash
claptrap_flag='true'
claptrap_color='false'
```
