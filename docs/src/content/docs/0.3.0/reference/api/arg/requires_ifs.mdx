# `requires_ifs` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.requires_ifs" icon="open-book" variant="minimal" />

<Badge text="since 0.1.0" size="small" />

Allows multiple conditional requirements.

# Examples

## Example 1

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
    ```toml
    name = "prog"
    [args]
    config = { long = "config", action = "set", requires-ifs = [{ arg = "opt", value = "special.conf" }, { arg = "other", value = "other.conf" }] }
    opt = { long = "option", action = "set" }
    other = {}
    ```
  </TabItem>

  <TabItem label="YAML">
    ```yaml
    name: prog
    args:
      config:
        long: config
        action: set
        requires-ifs:
        - arg: opt
          value: special.conf
        - arg: other
          value: other.conf
      opt:
        long: option
        action: set
      other: {}
    ```
  </TabItem>

  <TabItem label="JSON">
    ```json
    {
      "name": "prog",
      "args": {
        "config": {
          "long": "config",
          "action": "set",
          "requires-ifs": [
            {
              "arg": "opt",
              "value": "special.conf"
            },
            {
              "arg": "other",
              "value": "other.conf"
            }
          ]
        },
        "opt": {
          "long": "option",
          "action": "set"
        },
        "other": {}
      }
    }
    ```
  </TabItem>
</Tabs>

### Input

```bash
prog --config other.conf
```

### Output

```bash
error: the following required arguments were not provided:
  <other>

Usage: prog --config <config> <other>

For more information, try '--help'.
```
