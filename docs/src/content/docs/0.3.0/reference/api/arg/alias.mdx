# `alias` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.alias" icon="open-book" variant="minimal" />

<Badge text="since 0.1.0" size="small" />

Add an alias, which functions as a hidden long flag.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
    ```toml
    name = "prog"
    [args]
    test = { long = "alias", alias = "alias", action = "set" }
    ```
  </TabItem>

  <TabItem label="YAML">
    ```yaml
    name: prog
    args:
      test:
        long: alias
        alias: alias
        action: set
    ```
  </TabItem>

  <TabItem label="JSON">
    ```json
    {
      "name": "prog",
      "args": {
        "test": {
          "long": "alias",
          "alias": "alias",
          "action": "set"
        }
      }
    }
    ```
  </TabItem>
</Tabs>

### Input

```bash
prog --alias cool
```

### Output

```bash
claptrap_test='cool'
```
