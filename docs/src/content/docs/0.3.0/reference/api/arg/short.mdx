# `short` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.short" icon="open-book" variant="minimal" />

<Badge text="since 0.1.0" size="small" />

Sets the short version of the argument without the preceding `-`.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
    ```toml
    name = "prog"
    [args]
    config = { short = 'c', action = "set" }
    ```
  </TabItem>

  <TabItem label="YAML">
    ```yaml
    name: prog
    args:
      config:
        short: c
        action: set
    ```
  </TabItem>

  <TabItem label="JSON">
    ```json
    {
      "name": "prog",
      "args": {
        "config": {
          "short": "c",
          "action": "set"
        }
      }
    }
    ```
  </TabItem>
</Tabs>

### Input

```bash
prog -c file.toml
```

### Output

```bash
claptrap_config='file.toml'
```
