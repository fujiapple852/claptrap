# `raw` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.raw" icon="open-book" variant="minimal" />

<Badge text="since 0.1.0" size="small" />

Consume all following arguments.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
    ```toml
    name = "prog"
    [args]
    everything = { raw = true }
    ```
  </TabItem>

  <TabItem label="YAML">
    ```yaml
    name: prog
    args:
      everything:
        raw: true
    ```
  </TabItem>

  <TabItem label="JSON">
    ```json
    {
      "name": "prog",
      "args": {
        "everything": {
          "raw": true
        }
      }
    }
    ```
  </TabItem>
</Tabs>

### Input

```bash
prog -- --foo something -- -v -v -v -b -b -b --baz -q -u -x
```

### Output

```bash
claptrap_everything=('--foo' 'something' '--' '-v' '-v' '-v' '-b' '-b' '-b' '--baz' '-q' '-u' '-x')
```
