# `display_order` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.display_order" icon="open-book" variant="minimal" />

<Badge text="since 0.1.0" size="small" />

Allows custom ordering of args within the help message.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
    ```toml
    name = "prog"
    [args]
    boat = { short = 'b', long = "boat", action = "set", display-order = 0, help = "Some help and text" }
    airplane = { short = 'a', long = "airplane", action = "set", display-order = 0, help = "I should be first!" }
    custom-help = { short = '?', action = "help", display-order = 100, help = "Alt help" }
    ```
  </TabItem>

  <TabItem label="YAML">
    ```yaml
    name: prog
    args:
      boat:
        short: b
        long: boat
        action: set
        display-order: 0
        help: Some help and text
      airplane:
        short: a
        long: airplane
        action: set
        display-order: 0
        help: I should be first!
      custom-help:
        short: '?'
        action: help
        display-order: 100
        help: Alt help
    ```
  </TabItem>

  <TabItem label="JSON">
    ```json
    {
      "name": "prog",
      "args": {
        "boat": {
          "short": "b",
          "long": "boat",
          "action": "set",
          "display-order": 0,
          "help": "Some help and text"
        },
        "airplane": {
          "short": "a",
          "long": "airplane",
          "action": "set",
          "display-order": 0,
          "help": "I should be first!"
        },
        "custom-help": {
          "short": "?",
          "action": "help",
          "display-order": 100,
          "help": "Alt help"
        }
      }
    }
    ```
  </TabItem>
</Tabs>

### Input

```bash
prog --help
```

### Output

```bash
Usage: prog [OPTIONS]

Options:
  -a, --airplane <airplane>  I should be first!
  -b, --boat <boat>          Some help and text
  -?                         Alt help
  -h, --help                 Print help
```
