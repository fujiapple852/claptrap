# `required` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.required" icon="open-book" variant="minimal" />

<Badge text="since 0.1.0" size="small" />

Specifies that the argument must be present.

# Examples

## Example 1

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
    ```toml
    name = "prog"
    [args]
    cfg = { long = "config", required = true, action = "set" }
    ```
  </TabItem>

  <TabItem label="YAML">
    ```yaml
    name: prog
    args:
      cfg:
        long: config
        required: true
        action: set
    ```
  </TabItem>

  <TabItem label="JSON">
    ```json
    {
      "name": "prog",
      "args": {
        "cfg": {
          "long": "config",
          "required": true,
          "action": "set"
        }
      }
    }
    ```
  </TabItem>
</Tabs>

### Input

```bash
prog --config file.conf
```

### Output

```bash
claptrap_cfg='file.conf'
```
