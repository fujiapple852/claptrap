# `required_unless_present` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.required_unless_present" icon="open-book" variant="minimal" />

<Badge text="since 0.1.0" size="small" />

Set this arg as required as long as the specified argument is not present at runtime.

# Examples

## Example 1

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
    ```toml
    name = "prog"
    [args]
    cfg = { long = "config", required-unless-present = "dbg", action = "set" }
    dbg = { long = "debug", action = "set-true" }
    ```
  </TabItem>

  <TabItem label="YAML">
    ```yaml
    name: prog
    args:
      cfg:
        long: config
        required-unless-present: dbg
        action: set
      dbg:
        long: debug
        action: set-true
    ```
  </TabItem>

  <TabItem label="JSON">
    ```json
    {
      "name": "prog",
      "args": {
        "cfg": {
          "long": "config",
          "required-unless-present": "dbg",
          "action": "set"
        },
        "dbg": {
          "long": "debug",
          "action": "set-true"
        }
      }
    }
    ```
  </TabItem>
</Tabs>

### Input

```bash
prog --debug
```

### Output

```bash
claptrap_dbg='true'
```
