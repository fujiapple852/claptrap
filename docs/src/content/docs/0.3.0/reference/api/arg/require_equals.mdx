# `require_equals` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Arg.html#method.require_equals" icon="open-book" variant="minimal" />

<Badge text="since 0.1.0" size="small" />

Requires that options use the `--option=val` syntax

# Examples

## Example 1

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
    ```toml
    name = "prog"
    [args]
    cfg = { long = "config", action = "set", require-equals = true }
    ```
  </TabItem>

  <TabItem label="YAML">
    ```yaml
    name: prog
    args:
      cfg:
        long: config
        action: set
        require-equals: true
    ```
  </TabItem>

  <TabItem label="JSON">
    ```json
    {
      "name": "prog",
      "args": {
        "cfg": {
          "long": "config",
          "action": "set",
          "require-equals": true
        }
      }
    }
    ```
  </TabItem>
</Tabs>

### Input

```bash
prog --config=file.conf
```

### Output

```bash
claptrap_cfg='file.conf'
```
