# `disable_colored_help` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Command.html#method.disable_colored_help" icon="open-book" variant="minimal" />

<Badge text="since 0.1.0" size="small" />

Disables colorized help messages.

# Examples

## Example 1

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
    ```toml
    name = "myprog"
    disable-colored-help = true
    [args]
    cfg = { long = "config" }
    ```
  </TabItem>

  <TabItem label="YAML">
    ```yaml
    name: myprog
    disable-colored-help: true
    args:
      cfg:
        long: config
    ```
  </TabItem>

  <TabItem label="JSON">
    ```json
    {
      "name": "myprog",
      "disable-colored-help": true,
      "args": {
        "cfg": {
          "long": "config"
        }
      }
    }
    ```
  </TabItem>
</Tabs>

### Input

```bash
myprog --help
```

### Output

```bash
Usage: myprog [OPTIONS]

Options:
      --config <cfg>  
  -h, --help          Print help
```
