# `author` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Command.html#method.author" icon="open-book" variant="minimal" />

<Badge text="since 0.1.0" size="small" />

Sets the author(s) for the help message.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
    ```toml
    name = "myprog"
    author = "Me, me@mymain.com"
    help-template = "{name} ({version}) {author} - {usage}"
    [args]
    ```
  </TabItem>

  <TabItem label="YAML">
    ```yaml
    name: myprog
    author: Me, me@mymain.com
    help-template: '{name} ({version}) {author} - {usage}'
    args: {}
    ```
  </TabItem>

  <TabItem label="JSON">
    ```json
    {
      "name": "myprog",
      "author": "Me, me@mymain.com",
      "help-template": "{name} ({version}) {author} - {usage}",
      "args": {}
    }
    ```
  </TabItem>
</Tabs>

### Input

```bash
myprog --help
```

### Output

```bash
myprog () Me, me@mymain.com - myprog
```
