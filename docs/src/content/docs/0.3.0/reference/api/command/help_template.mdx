# `help_template` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Command.html#method.help_template" icon="open-book" variant="minimal" />

<Badge text="since 0.1.0" size="small" />

Sets the help template to be used, overriding the default format.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
    ```toml
    name = "myprog"
    help-template = "{before-help}{name} {version}\n{author-with-newline}{about-with-newline}\n{usage-heading} {usage}\n\n{all-args}{after-help}"
    [args]
    ```
  </TabItem>

  <TabItem label="YAML">
    ```yaml
    name: myprog
    help-template: |-
      {before-help}{name} {version}
      {author-with-newline}{about-with-newline}
      {usage-heading} {usage}

      {all-args}{after-help}
    args: {}
    ```
  </TabItem>

  <TabItem label="JSON">
    ```json
    {
      "name": "myprog",
      "help-template": "{before-help}{name} {version}\n{author-with-newline}{about-with-newline}\n{usage-heading} {usage}\n\n{all-args}{after-help}",
      "args": {}
    }
    ```
  </TabItem>
</Tabs>

### Input

```bash
myprog --help
```

### Output

```bash
myprog 

Usage: myprog

Options:
  -h, --help  Print help
```
