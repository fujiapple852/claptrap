# `subcommand_help_heading` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Command.html#method.subcommand_help_heading" icon="open-book" variant="minimal" />

<Badge text="since 0.1.0" size="small" />

Sets the help heading used for subcommands when printing usage and help.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
    ```toml
    name = "myprog"
    subcommand-help-heading = "Things"
    [[subcommands]]
    name = "sub"
    ```
  </TabItem>

  <TabItem label="YAML">
    ```yaml
    name: myprog
    subcommand-help-heading: Things
    subcommands:
    - name: sub
    ```
  </TabItem>

  <TabItem label="JSON">
    ```json
    {
      "name": "myprog",
      "subcommand-help-heading": "Things",
      "subcommands": [
        {
          "name": "sub"
        }
      ]
    }
    ```
  </TabItem>
</Tabs>

### Input

```bash
myprog --help
```

### Output

```bash
Usage: myprog [COMMAND]

Things:
  sub   
  help  Print this message or the help of the given subcommand(s)

Options:
  -h, --help  Print help
```
