# `after_help` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Command.html#method.after_help" icon="open-book" variant="minimal" />

<Badge text="since 0.1.0" size="small" />

Free-form help text for after auto-generated short help (`-h`).

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
    ```toml
    name = "myprog"
    after-help = "Does really amazing things for great people... but be careful with -R!"
    [args]
    ```
  </TabItem>

  <TabItem label="YAML">
    ```yaml
    name: myprog
    after-help: Does really amazing things for great people... but be careful with -R!
    args: {}
    ```
  </TabItem>

  <TabItem label="JSON">
    ```json
    {
      "name": "myprog",
      "after-help": "Does really amazing things for great people... but be careful with -R!",
      "args": {}
    }
    ```
  </TabItem>
</Tabs>

### Input

```bash
myprog -h
```

### Output

```bash
Usage: myprog

Options:
  -h, --help  Print help

Does really amazing things for great people... but be careful with -R!
```
