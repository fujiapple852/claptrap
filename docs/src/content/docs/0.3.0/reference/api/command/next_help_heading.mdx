# `next_help_heading` <LinkButton href="https://docs.rs/clap/latest/clap/struct.Command.html#method.next_help_heading" icon="open-book" variant="minimal" />

<Badge text="since 0.1.0" size="small" />

Set the default section heading for future args.

# Example

### Spec

<Tabs syncKey="format">
  <TabItem label="TOML">
    ```toml
    name = "prog"
    next-help-heading = "Configuration Options"
    [args]
    cfg = { long = "config", help = "Some help text describing the --config arg" }
    ```
  </TabItem>

  <TabItem label="YAML">
    ```yaml
    name: prog
    next-help-heading: Configuration Options
    args:
      cfg:
        long: config
        help: Some help text describing the --config arg
    ```
  </TabItem>

  <TabItem label="JSON">
    ```json
    {
      "name": "prog",
      "next-help-heading": "Configuration Options",
      "args": {
        "cfg": {
          "long": "config",
          "help": "Some help text describing the --config arg"
        }
      }
    }
    ```
  </TabItem>
</Tabs>

### Input

```bash
prog --help
```

### Output

```bash
Usage: prog [OPTIONS]

Options:
  -h, --help  Print help

Configuration Options:
      --config <cfg>  Some help text describing the --config arg
```
