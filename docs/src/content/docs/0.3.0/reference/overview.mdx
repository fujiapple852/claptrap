This section provides complete reference documentation for Claptrap.

### CLI Reference

The [CLI reference](/0.3.0/reference/cli) provides a complete list of the command line options available for Claptrap. This
information is available via the `--help` command line option and also in the man page on supported platforms.

### API Reference

The [API Reference](/0.3.0/reference/api) provides a reference of the Clap APIs supported by Claptrap. This includes
detailed documentation for each API, including examples.

### Shell Reference

The [Shell Reference](/0.3.0/reference/shell) provides a reference of how to use Claptrap from each shell supported by Claptrap.
